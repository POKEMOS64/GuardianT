# GuardianT Android App

Secure mobile client for the GuardianT ecosystem.

## Features

*   **Secure Pairing:** Connects to the GuardianT Hardware Key via Wi-Fi.
*   **Encrypted Chat:** End-to-end encrypted messaging interface.
*   **File Vault:** "Burn-on-read" file transfer and storage.
*   **No Cloud Dependency:** Works directly with your self-hosted server and hardware key.

## Build Instructions

1.  Open this folder in **Android Studio**.
2.  Wait for Gradle sync to complete.
3.  Build and Run on your Android device.

*Note: This project does not use Firebase Cloud Messaging to ensure privacy. It uses a polling mechanism to fetch messages from the server.*

# Приложение GuardianT для Android

Защищенный мобильный клиент для экосистемы GuardianT.

## Возможности

*   **Безопасное сопряжение:** Подключается к аппаратному ключу GuardianT через Wi-Fi.
*   **Зашифрованный чат:** Интерфейс для обмена сообщениями со сквозным шифрованием.
*   **Файловое хранилище:** Передача и хранение файлов с функцией "сжигания после прочтения".
*   **Без облачных зависимостей:** Работает напрямую с вашим собственным сервером и аппаратным ключом.

## Инструкция по сборке

1.  Откройте эту папку в **Android Studio**.
2.  Дождитесь завершения синхронизации Gradle.
3.  Соберите и запустите приложение на вашем Android устройстве.

*Примечание: Этот проект не использует Firebase Cloud Messaging для обеспечения конфиденциальности. Используется механизм опроса (polling) для получения сообщений с сервера.*
